# CERTEZA App - Autenticación y Roles

Este proyecto implementa autenticación con Microsoft (Outlook/Office365) usando NextAuth.js (v5) y un sistema de roles integrado con Postgres (Neon).

## Componentes

- Autenticación con Azure AD
- Adapter Postgres en Neon
- Middleware y páginas protegidas por roles
- UI de administración de roles
- Validación con Zod
- Pruebas unitarias con Vitest
